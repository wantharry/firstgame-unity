# Unity Asset Store Dependencies

These assets are NOT included in this repository (too large).
To restore them: Unity → Window → Package Manager → My Assets → find & import each one.

| Asset Name              | Folder in project            | Notes                              |
|-------------------------|------------------------------|------------------------------------|
| Furnished Cabin         | Assets/FurnishedCabin/       | Main cabin building & interior     |
| Grass Flowers FREE (ALP_Assets) | Assets/ALP_Assets/   | Grass & flower environment details |

## Unity Package Manager packages (auto-restored on open)

These are listed in `Packages/manifest.json` and restored automatically by Unity:

- Universal Render Pipeline (URP) 17.5.0
- Input System 1.19.0
- AI Navigation 2.0.13
- ProBuilder 6.1.2
- Cinemachine 3.1.7
- Terrain Tools 5.3.2

## How to re-import Asset Store assets after cloning

1. Open the project in Unity (packages restore automatically)
2. Go to **Window → Package Manager**
3. Change the dropdown to **My Assets**
4. Search for **Furnished Cabin** → Download → Import
5. Search for **Grass Flowers FREE** → Download → Import
